Tips When Writing HTML

Use Emmet

Emmet is a popular extension that most IDEs have nowadays. It helps you write HTML faster by way of shorthands.

Keep It Clean

Avoid nesting elements if it's not necessary. This happens a lot with block elements and we don't even realize it.

Use Shortcuts

If your IDE has features that would allow for faster writing, use it. A lot of editor has a duplicate feature which allow you to copy and paste code with just a few key strokes.

Commonly Used Tags

These are the tags you'll mostly see in all HTML pages.

  • div

    The div defines a section element. It is the universal element. This means that you can build a website with just div. In fact, that's how it used to be but nowadays, it is commonly used as a container for other elements.

  • h1, h2, h3, h4, h5, h6

    The h1 to h6 defines the HTML headings from the largest to the smallest. It is used to give titles to sections. The sizes are there to help indicate the importance of a section.

  • p

    The p tag defines a paragraph. It really is just a div for text wit margins (spacing).

  • ul and li

    The ul tag defines an unordered or bulleted list. I does what it says. It lists items but it works not only with text but also with other elements.

  • a

    An a tag is used to define hyperlinks which used to link from one page to the other.

  • button

    A button tag does exactly what it says. It creates a button but unlike a link, by default it doesn't do anything.

  • input

    An input tags define an element that holds different data. It can be anything, from a simple text field, to checkboxes and buttons.

  • form

    A form tag defines an element that sends a request, typically your input data, to the backend.

Commonly Used Attributes

  • class

    The class attribute is used to specify one or more elements. This is great when you want to assign styles to multiple elements at once.

  • id

    The id attribute gives the element a unique identifier that the CSS and JavaScript can refer to.

  • name

    The name attribute specify a name for an element. It has different uses depending on the element but its main use is for reference to the data that is submitted from the form.
